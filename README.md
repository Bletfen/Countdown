# ⏳ Countdown Timer

A simple countdown timer built using HTML, CSS, and JavaScript.  
This project counts down to a specified date and time, updating every second.

---

## Technologies Used

- **HTML** – for structure
- **CSS** – for styling
- **JavaScript** – for countdown logic and interactivity

---
